# 如何表示一张图 G=(V,E)

## 1、邻接链表

### 定义:

- 由一个包含|V|条链表的数组Adj所构成 无向图O(V+2E) 有向图O(V+E)
- 鲁棒性很高(不同的图(无向图到有向图以及权重图)容易修改)
- 不能快速判断任意两个节点是否有边相连

### 用途:

通常表示稀疏图(|E|远小于|V|^2)
